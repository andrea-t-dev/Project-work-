Realizzazione di un programma in Python che permetta di effettuare una previsione di raccolto stagionale.


IMPORTANTE: una volta scaricato il programma si raccomanda di avviarlo tramite Powershell o CMD. Windows non gestisce correttamente la creazione del grafico avviandolo con un semplice doppio click e utilizzando l'IDE di Python.

ESEMPIO di avvio da Powershell o CMD: python.exe Raccolto_stagionale.py
